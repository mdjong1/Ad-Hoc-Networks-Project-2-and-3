# MACAW and AODV Protocol Demo
This repository contains demo's for the MACAW MAC protocol as well as the AODV Routing protocol.
It was created as a university project for the Delft University of Technology Ad-Hoc Networks course.
All code is licensed as MIT and is therefore free to (re-)use in any further projects and demo's.

This would not have been possible without the work of [cjaikaeo](https://gitlab.com/cjaikaeo) in
[WSNSimPy](https://gitlab.com/cjaikaeo/wsnsimpy).
